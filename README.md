Racecar Simulator
=================
Simulates an RC car in the MIT tunnel.

1. Copy meshes/mit_tunnel_layout and meshes/racecar in /home/ubuntu/.gazebo/models folder 

2. Run "roslaunch racecar_gazebo racecar_tunnel.launch"
   - if Gazebo crashes, run standalone gazebo in a terminal and load the models so that it is stored in cache
   - run the above roslaunch command
